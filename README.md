# Tic Tac Toe Game

## Project Overview

This project is a console-based Tic Tac Toe game developed using Java. The game allows two players to play alternately by entering their moves through the keyboard. The program checks for valid moves, updates the game board, and determines the winner.

## Features

- Two-player gameplay
- Interactive console interface
- Move validation
- Winner detection
- Draw condition handling
- Easy-to-understand game logic

## Technologies Used

- Java
- VS Code
- Scanner Class
